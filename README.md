## 前端项目结构
```
src/
├── App.vue                 # 根组件
├── assets/                 # 静态资源
│   └── logo.svg
├── components/            # 公共组件
│   ├── FAQ.vue           # FAQ组件
│   └── Testimonials.vue  # 用户评价组件
├── i18n/                 # 国际化
│   ├── index.js         # i18n配置
│   └── langs/           # 语言文件
│       ├── zh-CN.js
│       ├── en-US.js
│       └── ...
├── pages/               # 页面组件
│   └── home/           
│       └── index.vue   # 首页组件
└── router/             # 路由配置
    └── index.js
```
